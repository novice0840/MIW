Perlin Noise 시각화

위키피디아 Perlin Noise 문서의 핵심 개념을 인터랙티브하게 탐험

1. Gradient Grid & Dot Product

격자 교차점마다 랜덤 gradient 벡터(화살표)를 배치하고,
각 점에서 4개 꼭짓점과의 dot product를 계산합니다.

2. Fade 함수 비교

원래 Perlin: 3t²−2t³ (빨간) vs Improved: 6t⁵−15t⁴+10t³ (초록)
Improved 버전은 2차 도함수까지 연속이라 경계가 더 부드럽습니다.

3. 1D Perlin Noise

fade 함수 선택에 따른 1D noise 결과 비교.
Improved(초록)가 격자 경계에서 더 부드럽게 연결됩니다.

4. 2D Perlin Noise

2D noise를 흑백 이미지로 시각화. scale을 조절해보세요.

5. fBm (Fractal Brownian Motion)

여러 옥타브를 겹쳐 디테일을 추가합니다.
낮은 옥타브 = 큰 형태, 높은 옥타브 = 세밀한 요철.

6. 지형 생성 응용

fBm 결과를 높이맵으로 사용해 마인크래프트식 지형을 생성합니다.
높이에 따라 물/모래/잔디/돌/눈이 결정됩니다.
